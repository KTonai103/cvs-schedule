# 心臓血管外科 電話帳アプリ

## 公開URL

https://ktonai103.github.io/cvs-schedule/pb-a280c8216931a803/

## 構成

- `index.html` — 単一HTMLファイル（データ・UI・検索ロジックすべて内包）

## 元データの所在

| データ | ファイル |
|--------|---------|
| 医師・スタッフ・共用番号 | `資料/心臓外科連絡先一覧.md` / `資料/PHS番号表.md` |
| 医師個人番号 | `資料/心臓外科個人番号表.md` |
| PHS外線番号（PDF原本） | `資料/PHS番号表（外線）.pdf` |
| 院内重要連絡先（74件） | 手配師PHSに登録されていた内線番号から抽出 |

統合済みの一覧は `資料/心臓外科連絡先一覧.md` に集約。

## データ構造（index.html内）

```javascript
const DATA = {
  doctors: [ ... ],  // 医師（卒年・PHS内線・外線・個人番号）
  staff:   [ ... ],  // スタッフ（PHS内線・外線）
  shared:  [ ... ]   // 院内連絡先（内線のみ）
};
```

各エントリの共通フィールド:
- `name` — 表示名
- `reading` — ひらがな読み（検索用）
- `phs` — 内線番号
- `dial` — PHS外線（院内連絡先は空文字）

医師のみ追加フィールド:
- `year` — 卒年
- `personal` — 個人電話番号

## 更新手順

1. `資料/心臓外科連絡先一覧.md` を最新にする
2. `index.html` の `DATA` オブジェクト内の該当配列を編集
3. 読み（`reading`）はひらがな検索用なので必ず付与する
4. commit & push → GitHub Pagesに自動反映
